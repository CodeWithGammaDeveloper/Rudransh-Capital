# Rudransh Capital

A premium, responsive corporate website for Rudransh Capital built with pure HTML, CSS, and JavaScript.

## Features
- Fully responsive layout for mobile, tablet, and desktop
- Premium financial services landing experience
- Animated counters, reveal effects, testimonial carousel, FAQ accordion, and dark mode
- Accessible contact form with client-side validation and success modal
- Sticky navigation, scroll progress bar, floating action buttons, live chat UI, and cookie banner

## Structure
- index.html – main page structure and sections
- style.css – premium visual design, layout, and responsive styling
- script.js – interactivity, animations, validation, and UI behavior
- assets/images – imagery for the site
- assets/icons – icons and branding assets

## Run Locally
Open index.html in your browser or serve the folder with any static server.

Example:
```bash
python -m http.server 8000
```
Then visit http://localhost:8000
